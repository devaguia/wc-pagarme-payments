== Description ==
Payment plugin for WooCommerce/WordPress using the Pagar.me API

= Requisitos =

- The PHP version has to be a less 7.4.
- WooCommerce version 5.0.x ou maior.

== Installation ==
1. Upload this plugin in your WordPress site, and activate it;
2. Access on the side menu "WooCommerce -> Pagar.me Settings;
3. Configure your Pagar.me credentials;
4. Configure the webhooks on your Pagar.me panel using your "Webhook Endpoint" link.
