Pagar.me pagamentos para WooCommerce

Sobre o plugin

O plugin para WordPress Pagar.me pagamentos para WooCommerce tem como objetivo adaptar o serviço de pagamento da Pagar.me como um método de pagamento que integre com o plugin WooCommerce. O plugin utiliza a API pública disponibilizada pela própria pagadora em sua documentação.
Este não é um plugin oficial e não possui vínculo com a empresa Pagar.me.

Configurações do plugin

Status

Essa configurações tem como objetivo controlar qual o status será usado para representar uma um pedido bem sucedido no WooCommerce, para os métodos de pagamento disponibilizados pelo plugin.

Exportar configurações

Com essa configuração você consegue exportar um arquivo com dados relacionados as configurações do plugin Pagar.me pagamentos para WooCommerce, WooCommerce e do WordPress.

Apagar configurações

Ao marcar essa opção, você concorda que todos os dados do plugin serão apagados ao desativá-lo.

Endpoint de webhook

O link presente no campo de "Endpoint de webhook" é o link que será utilizado para poder configurar as notificações que o plugin irá receber da Pagar.me.

Opções de pagamentos

Os dois campos presentes nessa configuração se referem as chaves privadas e púbilicas disponibilizadas pela Pagar.me e utilizadas para realizar a conexão com a API da pagadora.
Para acessar essas chaves, dentro do painel da Pagar.me vá em:
Configurações -> Chaves

Configurações de Webhook

Onde configurar?

Para configurar os eventos de Webhook, dentro do painel da Pagar.me vá em: Configurações -> Webhooks -> + Criar Webhook.

URL

Use a url gerada pelo plugin nas configurações(Endpoint de webhook) para preencher o campo de URL.

Máximo de tentativas

Recomendamos que mantenha essa campo preenchido com 3 tentativas.

Eventos

Para essa integração, nós só vamos utilizar os eventos referentes aos pedidos. Então clique na opção "Pedidos", e marque os eventos abaixo:
  • order.canceled
  • order.paid
  • order.payment_failed
